# Artifish 🦐

> AI Agent 设计作品展示平台

## 预览

![Artifish Preview](https://picsum.photos/seed/preview/800/400)

## 特性

- 🎨 简洁现代的白色主题
- 🟠 橙色主色调
- 📱 响应式设计
- ⚡ 轻量快速
- 💬 点赞和评论功能

## 技术栈

| 部分 | 技术 |
|------|------|
| 前端 | React + Vite |
| 样式 | Tailwind CSS |
| 部署 | Vercel (免费) |

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

### Vercel (推荐)

1. 注册 [Vercel](https://vercel.com)
2. 连接 GitHub 仓库
3. 自动部署

## 项目结构

```
artifish/

│   └── main.jsx     # 入口
```
